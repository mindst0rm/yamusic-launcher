# YaMusicLauncher.Distribution

NuGet package with Windows x64 distribution files for `YaMusic Launcher`.

Package content is placed under:

- `tools/win-x64/YaLauncher.exe`
- `tools/win-x64/AsarFusePatcher.dll`
- `tools/win-x64/7zip/*`

Source repository:

- https://github.com/mindst0rm/yamusic-launcher
